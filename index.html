Personal Access Token Tutorial

Generating a Personal Access Token

Please follow the steps located here! . If these do not work for you, please follow the steps located below.

Follow the commands in order:

  • Open Terminal (Mac) or Command Prompt (Windows)
  • cd into Promineo_Tech/Week-01-CLI_Source_Control_and_Variables directory
  • Create a new repository on GitHub by clicking the New button on the top right of the screen.
  • Name the repository some-repo-name-here and click Create Repository
  • Copy the portion of code located under, ...or create a new repository on the command line.
  • The code will look something like:
    • echo "# github-tutorial" >> README.md
    • git init
    • git add README.md
    • git commit -m "first commit"
    • git branch -M main
    • git remote add origin https://github.com/your_user/your_repo
    • git push -u origin main
  • Paste the code into the terminal/command prompt and press enter.
  • It may prompt you to enter your GitHub email and password.
    • git config --global user.email your_github_email@gmail.com
    • git config --global user.name your_github_username
  • You may need to run the git push -u origin main again.
  • Go to your GitHub repository and you should see your README.md file.
  • Press enter and you should see a message that says, "Everything up to date"

    • Secure Shell (SSH) Tutorial

    Part 1: Generating your SSH token

    Follow the commands in order:

  • Open Terminal
  • cd ~/
  • ls and look for a folder called .ssh
    • If you do not have a folder called .ssh run mkdir .ssh
  • cd .ssh
  • ssh-keygen -t ed25519 -C " your-github-email@example.com "
  • Enter a file in which to save the key: id_rsa
  • Enter passphrase (empty for no passphrase): Press Enter
  • Enter same passphrase again: Press Enter
  • Open .
  • Open id_rsa.pub with textEdit and copy the code inside.

    • Part 2: Adding your SSH token to GitHub

  • In the upper-right corner of any page, click your profile photo, then click Settings.
  • In the "Access" section of the sidebar, click SSH and GPG keys.
  • Click New SSH key or Add SSH key.
  • In the "Title" field, add a descriptive label for the new key such as, id_rsa
  • Paste your SSH token key you copied earlier into the Key field.
  • Click Add SSH key.
  • If prompted, confirm your GitHub password.

    Part 2: Saving your workspace

  • Click on the File icon on the top of the screen.
  • Click on the Add Folder to Workspace icon on the top of the screen.
  • Navigate to the folder you want to add to your workspace and click Open.
  • You should see the folder you added to your workspace in the Explorer section.
  • Click on the File icon on the top of the screen.
  • Click on the Save Workspace As icon on the top of the screen.
  • Name your workspace and click Save.
  • You should see the folder you added to your workspace in the Explorer section.
  • Right-click on the folder you wish to connect your GitHub repo to.
  • Click on the Open in Integrated Terminal option.
  • You should see a terminal open up at the bottom of your screen.
  • At this point you can follow the steps from the following tutorials.
